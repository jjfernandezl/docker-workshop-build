# Commands help file

This file provides a list of commands that will be used during the workshop, so they can be copied in the terminal if needed

## Previous steps

The workshop will utilize several official Docker images from Docker Hub. To save time for varying network conditions on the day of the workshop, it is highly recommended that you pull these images in advance.
Execute the following commands in your terminal:

```bash
docker pull postgres:18
docker pull node
docker pull node:20.19.5-alpine3.22
docker pull ubuntu:22.04
```
